# ubuntu_novnc

source code of light weight xubuntu and lubuntu docker images. Each only requires 1GB memory and 1-2 CPUs to run

Two flavors. Both use novnc, with resolution 1600x900

1. xubuntu - based on xubuntu with xfce4 WM,
2. lubuntu - based on lubuntu with lqxt WM.

Makefile builds images. run_xfce.sh or run_lxqt.sh to run. Tested on Apple silicon with colima.
